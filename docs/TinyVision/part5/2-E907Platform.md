# E907 平台

玄铁E907 是一款完全可综合的高端 MCU 处理器。它兼容 RV32IMAC 指令集，提供可观的整型性能提升以及高能效的浮点性能。E907 的主要特性包括：单双精度浮点单元，以及快速中断响应。

![img](assets/post/README/E907特性.jpg)

在V85x平台中使用的E907为RV32IMAC，不包括 P 指令集。
